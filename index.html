Help

Browser requirement

iPhone Safari + Chrome do NOT support Web Bluetooth. You need Bluefy from the App Store ($2.99) — it's a WebKit browser with Web Bluetooth API enabled.

Open this URL in Bluefy:

Connection issues
  • Make sure the scooter is powered on and within 1 m of the iPhone
  • The scooter must NOT already be connected to another phone or app
  • If the device chooser shows nothing, force-quit Bluefy and retry
  • iOS Bluetooth must be enabled (Settings -> Bluetooth)
About the protocol

Service: 2600. Control char: 7000. Data char: 7001. SHA-256 image digest. Optional 64-byte signature. Sector-based upload (4096 bytes/sector) with NACK retransmission. See the GitHub repo for details.
